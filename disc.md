15) 1. Нет реальной рекомендательной логики
    Сейчас: формула (100 / наценка) × качество. Это детерминированный фильтр, не рекомендательная система.

    Нужно:
     - Контентная фильтрация (похожие товары по характеристикам)
     - Персонализация — если пользователь смотрит бюджетные GPU, рекомендовать бюджетные аналоги
     - «Горячее/холодное» — товары с лучшей ценой прямо сейчас

    2. Нет предсказания цен
    «Оценка цен» подразумевает не только текущую, но и будущую цену.

    Нужно:
     - Линейный тренд цены (растёт/падает)
     - Прогноз на 1-3 месяца
     - «Покупать сейчас» / «Подождать»

    3. Нет мульти-маркетплейс сравнения
    «На онлайн-площадках» — множественное число. Сейчас цены агрегированные.

    Нужно:
     - «Где купить дешевле»: DNS vs Ozon vs Citilink
     - Рекомендация конкретного магазина, не только товара

    4. Нет обратной связи
    Рекомендательная система должна учиться.

    Нужно:
     - «Полезно / не полезно» для каждой рекомендации
     - Отслеживание кликов — что пользователь реально выбирает
     - Адаптация рекомендаций на основе поведения

    5. Нет ML-компонента
    Для диплома со словом «система» — обычно ожидают модель, не только формулу.

    Нужно:
     - Простая регрессия для предсказания цен
     - Кластеризация товаров по сегментам
     - или хотя бы scikit-learn pipeline
